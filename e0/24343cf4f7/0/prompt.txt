ext to Audio Converter (Sarvam.ai)
This project converts text from a story.txt file into an audio file (audio.mp3) using the Sarvam.ai Text-to-Speech (TTS) API.

Prerequisites
Python 3.8+
Sarvam.ai API Key
How to get Sarvam.ai API Key
Sign Up: Go to dashboard.sarvam.ai and create an account.
API Keys: Navigate to the "API Keys" section in the sidebar.
Generate: Click "Create New API Key", give it a name, and copy the key immediately.
Credits: Check the "Billing" or "Usage" tab to ensure you have active free credits (usually provided to new accounts).
Setup (Recommended: Virtual Environment)
Since modern Linux distributions protect the system Python environment, use a virtual environment:

Create a virtual environment:

python3 -m venv venv
Activate the environment:

source venv/bin/activate
Install the dependencies:

pip install -r requirements.txt
Usage
Place your text in story.txt.
Ensure your virtual environment is active (source venv/bin/activate).
Run the script:
python main.py
The output will be saved as audio.mp3.
Audio Preview

Your browser does not support the audio element.
Features
Supports Indian English (en-IN), Telugu (te-IN), and other Indian languages.
Smart Chunking: Automatically splits long text files to stay within API limits.
